PAVITRA OS 1.0 — DOCUMENTATION INDEX
================================================================================

Welcome to Pavitra OS — a universal application platform built on Debian 12.

This directory contains the full documentation for Pavitra OS. Read the files
in the order listed below for the best understanding of the system.

FILES IN THIS DIRECTORY
------------------------

  README.txt              — This file. Start here.
  getting-started.txt     — First steps after installing Pavitra OS
  windows-compat.txt      — Running Windows apps via Wine
  macos-compat.txt        — Running macOS apps via Darling (experimental)
  linux-apps.txt          — Running Linux binaries, AppImages, and Flatpaks
  debian-packages.txt     — Installing .deb packages with gdebi
  app-runner.txt          — Using the Pavitra App Runner (drag-and-drop launcher)
  desktop-guide.txt       — XFCE4 desktop, panels, themes, and settings
  system-config.txt       — Hostname, users, firewall, updates, performance
  troubleshooting.txt     — Common problems and their solutions
  faq.txt                 — Frequently asked questions
  changelog.txt           — Version history and release notes

QUICK REFERENCE
---------------

  Run a Windows .exe :    run-windows-app /path/to/program.exe
  Run a macOS .app   :    run-macos-app /path/to/App.app
  Install a .deb     :    install-deb /path/to/package.deb
  Run an AppImage    :    appimage-run /path/to/App.AppImage
  Universal launcher :    pavitra-app-runner  (also in the dock)
