The System

Built on Debian 12 Bookworm with a 6.1 kernel. Pavitra OS is not a fork — it is a fully assembled live operating system that ships four complete application runtimes as first-class citizens.

The Wine compatibility layer handles Windows executables. Darling reimplements the macOS Darwin API without any Apple code. Native Debian packages and AppImages complete the fourth ecosystem. All four run simultaneously with no configuration required.

Compatibility

One machine.
Every application.

01
Windows
Run .exe files natively through Wine 9. Thousands of Windows applications work without configuration.
run-windows-app program.exe
02
macOS
Run .app bundles through Darling — an open-source Darwin API reimplementation. No Apple code.
run-macos-app App.app
03
Debian
Install any .deb package with one command or a double-click in Thunar file manager.
install-deb package.deb
04
AppImage
Drop any .AppImage and run instantly. No root, no installation, no system modifications.
appimage-run App.AppImage

Features

Engineered for the machine you actually use.

01

Pavitra App Runner

Drag any file onto the dock launcher. It auto-detects the format and launches the correct runtime.

02

XFCE4 Desktop

Lightweight and fast. Ships fully configured — Greybird-dark GTK, Papirus icons, Noto fonts. Ready from first boot.

03

Security by Default

UFW firewall on from first boot. Unattended security updates from Debian repositories only. No telemetry.

04

Flatpak + Flathub

GNOME Software ships pre-installed with Flathub configured. Access the full Flatpak ecosystem without a terminal.

05

Performance Tuning

zram compressed swap with LZ4. CPU schedutil governor. thermald for thermal management. Every boot is fast.

06

Full Application Suite

Firefox ESR, LibreOffice, GIMP, VLC, GParted, Thunar, Transmission — pre-installed and configured.

Open Source

Reproducible from source.

The entire pipeline is scripted with live-build. Clone and produce an identical 4.9GB bootable ISO on any Debian 12 host.

View on GitHub Download ISO
bash 
# Clone the repository
git clone https://github.com/brighteyekid/Pavitra-OS.git
cd Pavitra-OS

# Install build dependencies
sudo bash pavitra-os/setup-build-env.sh

# Build the live filesystem (~60-90 min)
sudo bash pavitra-os/build-base.sh

# Generate the bootable ISO
sudo bash pavitra-os/build-final-iso.sh
